# 成绩链系统文档
基于区块链智能合约的多模态成绩上链系统
## 文档使用polkadotjs


This is a documentation portal for the `@polkadot` family of tools and libraries. The latest version is always available at [https://polkadot.js.org/docs/](https://polkadot.js.org/docs/)


## 如何使用

```bash
# 确保安装node和npm  node>=16*

# 安装yarn(已安装可以忽略)
npm install -g yarn

# 安装依赖
yarn

# 本地启动文档调试
yarn start

# 打包
yarn build

```
